# 📁 Multi-file Chatbot

A modular AI chatbot application organized using multiple Python modules to improve maintainability and scalability.

---

# Overview

This project refactors the chatbot into a clean multi-file architecture by separating configuration, AI interaction, and file management into dedicated modules.

---

# Features

- Modular architecture
- Reusable components
- Groq API integration
- Configuration management
- File handling utilities

---

# Tech Stack

- Python
- Groq API
- JSON
- python-dotenv

---
