# 📰 AI News Summarizer

Fetches the latest news articles using NewsAPI and generates concise summaries using the Groq LLM.

---

# Overview

This project combines external REST APIs with AI summarization to produce readable summaries of current news.

---

# Features

- Fetch latest news
- AI-generated summaries
- NewsAPI integration
- Groq integration
- Error handling

---

# Tech Stack

- Python
- Requests
- NewsAPI
- Groq API
- JSON
